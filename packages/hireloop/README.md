# MCP HireLoop

Thin MCP wrapper for a running HireLoop backend service.

## Tools

- `hireloop.run_start`
- `hireloop.run_active`
- `hireloop.run_status`
- `hireloop.run_results`
- `hireloop.jobs_list`
- `hireloop.jobs_update`
- `hireloop.sheet_sync`
- `hireloop.profile_context`
- `hireloop.cv_render_plan`
- `hireloop.apply_start`
- `hireloop.apply_status`

## Quick start

```bash
HIRELOOP_BASE_URL=http://127.0.0.1:8787 npx @waleedyousaf07/mcp-hireloop@latest
```

See `setup.md` for local development notes.
